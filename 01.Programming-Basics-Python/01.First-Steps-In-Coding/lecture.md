# First Steps in Coding

This lecture introduces the fundamental principles of programming using Python. The goal is to build a basic understanding of how a program works and how instructions are executed.

---

## 📚 What is a Program?

A program is a sequence of instructions that a computer executes step by step.

In Python, programs are executed from top to bottom.

---

## 🖥 Input and Output (I/O)

One of the first concepts in programming is interacting with the user.

### Input

Allows reading data from the keyboard:

```python
name = input()
```

### Output

Allows displaying information on the screen:

```python
print(name)
```

---

## 🔤 Variables

Variables are used to store values.

```python
username = "Ivan"
age = 25
```

Basic data types:

* `str` (text)
* `int` (integer)
* `float` (decimal number)

---

## 🔢 Basic Calculations

Python allows performing arithmetic operations:

```python
a = 5
b = 3

sum = a + b
product = a * b
```

Main operators:

* `+` addition
* `-` subtraction
* `*` multiplication
* `/` division

---

## 🔄 Type Conversion (Casting)

Input from `input()` is always treated as text and must be converted when needed:

```python
number = int(input())
price = float(input())
```

---

The tasks in this lecture focus on:
